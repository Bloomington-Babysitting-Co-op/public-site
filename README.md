# Bloomington Babysitting Co-op - Public Site
Created for Cloudflare Pages.

## Setup
### Local Development
1. Clone this repo
2. Open `public/index.html` directly in a browser

## Deploy
1. Push changes to `main` branch to GitHub

### Initial Setup
1. Create a Cloudflare Pages project
   * Project name: `bbc-public`
   * Production branch: `main`
   * Build command: *(leave empty)*
   * Build output directory: `public`
   * Click "Save and Deploy"
