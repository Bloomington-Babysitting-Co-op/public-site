# Bloomington Babysitting Co-op - Public Site
Created for Cloudflare Pages.

## Setup
### Local Development
1. Clone this repo
2. Open `public/index.html` directly in a browser

## Deploy
1. Push changes to `main` branch to GitHub

### Initial Setup
1. Create a [Cloudflare Pages](https://dash.cloudflare.com/f4306531b5050e87dd2882f786d5c4ad/workers-and-pages) project
   * Project name: `bbc-public`
   * Production branch: `main`
   * Build command: *(leave empty)*
   * Build output directory: `public`
   * Click "Save and Deploy"
2. Set up a [Custom Domain](https://dash.cloudflare.com/f4306531b5050e87dd2882f786d5c4ad/pages/view/bbc-public/domains) linking `www.bloomingtonbabysitting.org` to the project
